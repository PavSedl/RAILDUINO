1. Connect the Railduino module using USB port to your PC.
2. Execute RAILDUINO HEX UPLOAD.bat file
3. In the command line write down the COM port which Railduino is connected to - eg. "COM4", hit enter
4. Then write the file name you want to download to Railduino - eg. "Railduino_1_3_LAN.hex", hit enter
5. After the download you should see statistics window with the message of succesfull download

1. Připojte Railduino modul k PC pomocí USB připojení
2. Spusťte soubor RAILDUINO HEX UPLOAD.bat
3. Do příkazového řádku napište USB port, ke kterému je modul připojen - např. "COM4", stiskněte enter
4. Napište jméno souboru, který chcete do modulu nahrát - např. "Railduino_1_3_LAN.hex", stiskněte enter
5. Po úspěšném nahrání se zobrazí okno se statistikami se zprávou o úspěšném nahrání
