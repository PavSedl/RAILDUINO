    Status LEDs

    BLUE LED

    - do not lit - no power supply, boot up phase or module failure
    - flashes 1 Hz - device operating OK
    - lit - awaiting IP address from the DHCP server

    GREEN LED

    - do not lit - no ongoing communication - device operating OK
    - flashes - device communicating
    - lit - communication failure
